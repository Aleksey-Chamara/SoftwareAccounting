> Тут описаны обязательные требования и хотелки для системы Учета ПО.

## Обязательные требования (MUST)

| **Требование** | **Описание** | **Критерий приёмки (Как проверить)** |
| :--- | :--- | :--- |
| **Учет компьютеров** | Возможность создания, редактирования и удаления карточки ПК (Владелец, MAC, Дата проверки). | <ol><li>Администратор заполняет форму.</li><li>Запись появляется в таблице `Computers`.</li><li>Данные выводятся на главной странице.</li></ol> |
| **Справочник ПО** | Ведение глобального списка разрешенных программ с указанием категории и типа лицензии. | <ol><li>Программа добавлена через админку.</li><li>Она становится доступна для выбора в выпадающем списке привязщика.</li></ol> |
| **Обработка софта "Other"** | Если ПО нет в справочнике, должна быть возможность вписать его название вручную (поле `CustomSoftwareName`). | <ol><li>В карточке ПК выбирается опция ручного ввода.</li><li>В БД создается запись, где `SoftwareId == null`.</li><li>В интерфейсе отображается маркер "Other: Название".</li></ol> |
| **Многоуровневая сортировка** | Сортировка реестра ПК по владельцам, MAC-адресам, ПО, типам лицензий и датам. | <ol><li>Клик по заголовку таблицы меняет порядок (по возрастанию/убыванию).</li><li>Сортировка происходит на уровне БД/контроллера.</li></ol> |
| **Технический паспорт** | Страница `Details`, собирающая всю информацию по ПК и списку софта (дата установки, лицензия). | <ol><li>Переход в детали ПК отображает структурированный лог.</li><li>Выводятся как системные, так и кастомные программы.</li></ol> |
| **Авторизация и роли** | Разделение прав. Обычный юзер только смотрит, Админ — редактирует. Упрощенный вход. | <ol><li>Вход под `user/user` скрывает кнопки "Создать/Удалить".</li><li>Вход под `admin/admin` открывает полный CRUD.</li></ol> |

## Желательные требования (NICE)

1. Экспорт технического паспорта компьютера в формат PDF или Excel для печати.
2. Автоматическая валидация формата MAC-адреса на фронтенде с помощью регулярного выражения.
3. Цветовая индикация компьютеров, у которых дата последней проверки просрочена более чем на 6 месяцев.
4. Счетчик общего количества использованных лицензий конкретного софта на главной странице справочника ПО.
